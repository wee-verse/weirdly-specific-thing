Privacy

Short version: there's no account system, no login, and none of the quizzes ask for personal information. Answers you give inside a quiz stay in your browser and are used only to pick which result to show you — they are not sent anywhere or stored on a server.

What this site currently collects

As of this page's last update, Weevil World does not run any analytics, advertising, or tracking scripts. The only data generated is standard web server/CDN logs (IP address, browser type, page requested) kept briefly by our hosting provider for security and reliability — the same baseline logging almost every website on the internet has, and not something we look at or use individually.

If that changes

If we add analytics or advertising in the future, this page will be updated first to describe exactly what's added, and — where required — a consent notice will appear on the site itself. We don't plan to sell or share personal data with third parties.

Cookies

This site does not currently set any cookies of its own.
